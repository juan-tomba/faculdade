# Listas Lineares

# Propriedades:
   *Pode ser Ordenada ou não*
     -Se for Ordenada, tem maior custo para inserir ou retirar, já Não Ordenada, a inserção é feita nas extremidades.
     -Temos a opção para o usuário definir se é ordenada ou não.

   *Sequencial ou Dinâmica*

   *Tem a opção de busca*
     -Com isso, temos a variável chave, que é um identificador.
     -Caso seja não ordenada, realizamos a busca sequencial (comparar uma a uma, até achar).
     -Caso contrário, podemos implementar outra busca (como busca binária).

   **Melhor algoritmo de ordenação é odd: N*Log(N)**
